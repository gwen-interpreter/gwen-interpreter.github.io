gwen

Behaviour driven automation
Feature
Drive web automation with feature specifcations like this (Gherkin):
  Feature: Google search

 Scenario: Lucky Google search
     Given I have Google in my browser                    (1)
      When I do a search for "Gwen automation"            (2)
      Then I should land on a Gwen page                   (3)
Meta
With automation glue in meta specifications like this (also Gherkin):
  Feature: Google search meta
                        
 @StepDef
 Scenario: I have Google in my browser                    (1)
     Given I start a new browser
      When I navigate to "http://www.google.com"
      Then the page title should be "Google"
                        
 @StepDef
 Scenario: I do a search for "<query>"                    (2)
     Given the search field can be located by name "q"
       And link 1 can be located by css selector ".r > a"
      When I enter "$" in the search field
       And I click link 1
      Then the page title should not be "Google"
                                               
 @StepDef
 Scenario: I should land on a Gwen page                   (3)
      Then the current URL should match regex ".+[G|g]wen.*"
Gwen
Launch Gwen to bind the feature to meta and execute!
