Specifications in

Given a Gherkin feature

Declarative feature specifications
  1. Write Gherkin specifications that describe feature requirements as behaviour.

When I write Gwen meta

Imperative meta specifications
  1. Write Gherkin specifications that describe automation bindings and step definitions.

Automation out

Then Gwen will automate

Executable specifications interpreter
  1. Launch Gwen to bind Gherkin feature specifications to meta and execute them.

Quick start
  • Download the latest gwen-workspace.zip
  • Extract the zip file into a directory
  • Open a terminal session in the directory
  • Run the following command(s) to install
    • Mac/Linux
      • chmod -R u+x .
      • ./gwen -b
    • Windows
      • gwen -b
  • Start using Gwen
