gwen

Automation without all the programming


Specifications in   >   Automation out
  • 1. Take a feature specification that describes behaviour.
  • Gherkin feature
  • 2. Write a Gherkin specification to describe automation.
  • Gwen meta
  • 3. Launch Gwen to bind the feature to meta and execute.
  • Gherkin interpreter

Quick start
  • Install Java 8+
  • Download gwen-workspace.zip
  • Extract zip into a directory
  • Open terminal in directory and run:
    Linux:
    chmod -R u+x gwen*
    ./gwen -b
    Win:
    gwen -b
  • Start using Gwen
