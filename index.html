Quick start
  • Install Java 8+
  • Download latest gwen-workspace.zip
  • Extract zip file into a directory
  • Open terminal in directory and run
    • Mac/Linux
      • chmod -R u+x gwen*
      • ./gwen -b
    • Windows
      • gwen -b
  • Start using Gwen
