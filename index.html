Get started

Specifications in

Given a Gherkin feature

Declarative feature specifications
Write Gherkin feature specifications to describe all requirements in terms of behavior.

When I write Gwen meta

Imperative meta specifications
Write Gherkin meta specifications to describe all bindings and step definitions for automation.

Automation out

Then Gwen will automate

Executable specifications interpreter
Launch Gwen to bind feature specifications to meta specifications and execute them to achieve automation.

Quick start
  • Download and extract the latest gwen-workspace.zip
  • Open a terminal session in the extracted directory
  • Run the following command(s) to install:
    • Mac/Linux:
      • chmod -R u+x .
      • ./gwen -b
    • Windows:
      • gwen -b
  • Start using Gwen
