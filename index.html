Quick start
  • Download latest gwen-workspace.zip
  • Extract zip file into a directory
  • Open a terminal session in directory
  • Run following command(s) to install
    • Mac/Linux
      • chmod -R u+x .
      • ./gwen -b
    • Windows
      • gwen -b
  • Start using Gwen
