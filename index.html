gwen

Automation without all the programming
Feature
Given a behavioural specification that describes a feature.
Gwen meta
Write a technical specification to describe the automation.
Executable specification
Launch Gwen to bind the feature to meta and execute.

Quick start
Install Java 8+
Download gwen-workspace.zip
Extract zip into a directory
Open terminal in directory & run:
Linux:
   chmod -R u+x gwen*
   ./gwen -b
Win:
   gwen -b
