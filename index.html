gwen

Automation bindings for Gherkin
Declarative feature
Take a feature specification that describes behaviour.
Imperative meta
Compose a meta specification to describe automation.
Executable specification
Launch Gwen to bind the feature to meta and execute.

Quick start
Install Java
Download gwen-workspace.zip
Extract zip into a directory
Open terminal in directory & run:
Linux:
   chmod -R u+x gwen*
   ./gwen -b
Win:
   gwen -b
