Get started

Specifications in

Given a Gherkin feature

Business specifications
Declarative Gherkin specifications describe feature requirements and expected behaviours.

When I write Gwen meta

Automation specifications
Imperative Gherkin specifications describe automation bindings and executable step definitions.

Automation out

Then Gwen will automate

Executable specifications
Gwen binds feature specifications to meta specifications and executes them to achieve automation.

Quick install
  • Download and extract the latest gwen-workspace.zip
  • Open a terminal to the extracted directory
  • Run the following command(s)
    • Mac/Linux
      • chmod -R u+x .
      • ./gwen -b
    • Windows
      • gwen -b
