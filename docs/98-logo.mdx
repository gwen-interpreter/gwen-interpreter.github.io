The Gwen logo consists of a [Lorenz attractor](https://en.m.wikipedia.org/wiki/Lorenz_system) that looks like a butterfly.

Gwen evaluates software applications by mapping [Gherkin](https://cucumber.io/docs/gherkin/reference/) steps to instructions that:

- Create context (_Givens_)
- Perform actions (_Whens_)
- Assert outcomes (_Thens_)

The cumulative result is a chain of instructions that execute in sequence to model cause and effect. If a change to the system disrupts the chain, then Gwen will detect the chaos that emerges.
