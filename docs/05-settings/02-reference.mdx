# Settings Reference

## Core Settings

### Assertions

<details id="gwen-assertion-mode">
  <summary class="setting">

  ```properties
  gwen.assertion.mode
  ```
  <p>Controls whether or not hard, soft or sustained assertions are enabled.</p>
  </summary>
  <p>Supported values:</p>
  <ul>
    <li><code>hard</code> (default) to halt feature execution on the first assertion error and report failure</li>
    <li><code>soft</code> to collect all assertion errors and continue execution whilst reporting failures</li>
    <li><code>sustained</code> to collect all assertion errors and continue execution without reporting failures</li>
  </ul>
  
  <Tabs
  groupId="config"
  defaultValue="properties"
  values={[
    {label: 'gwen.properties file', value: 'properties'}
  ]}>

  <TabItem value="properties">

  ```properties
    gwen.assertion.mode = hard
  ```

  </TabItem>

  </Tabs>
