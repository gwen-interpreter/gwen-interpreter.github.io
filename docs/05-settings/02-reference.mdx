# Settings Reference

## Core Settings

<details id="gwen-assertion-mode">
  <summary class="setting">

  ```properties
  gwen.assertion.mode
  ```
  <p>Controls whether or not hard, soft or sustained assertions are enabled.</p>
  </summary>
  <p>Supported values</p>
  <ul>
    <li><code>hard</code> to halt feature execution on the first assertion error and report failure</li>
    <li><code>soft</code> to collect all assertion errors and continue execution whilst reporting failures</li>
    <li><code>sustained</code> to collect all assertion errors and continue execution without reporting failures</li>
  </ul>
  <p>Default value</p>
  <ul>
    <li><code>hard</code></li>
  </ul>
  <Tabs
  groupId="config"
  defaultValue="properties"
  values={[
    {label: 'gwen.properties file', value: 'properties'}
  ]}>

  <TabItem value="properties">

  ```properties
    gwen.assertion.mode = hard
  ```

  </TabItem>

  </Tabs>

<details id="gwen-associative-meta">
  <summary class="setting">

  ```properties
  gwen.associative.meta
  ```
  <p>Controls whether or not same named feature and meta files in a directory (excluding file extensions) are to be exclusively associated with each other only.</p>
  </summary>
  <p>Supported values</p>
  <ul>
    <li><code>true</code> to enable association
      <ul>
        <li>Gwen will load a discovered meta file only if it has the same name and is in the same directory of the feature currently being executed.</li>
      </ul>
    </li>
    <li><code>false</code> to disable association
      <ul>
        <li>Gwen will unconditionally load every meta file discovered in the path to the feature being executed.</li>
      </ul>
    </li>
  </ul>
  <p>Default value</p>
  <ul>
    <li><code>false</code></li>
  </ul>
  <Tabs
  groupId="config"
  defaultValue="properties"
  values={[
    {label: 'gwen.properties', value: 'properties'}
  ]}>

  <TabItem value="properties">

  ```properties
    gwen.associative.meta = false
  ```

  </TabItem>

  </Tabs>
