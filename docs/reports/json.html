JSON Reports

Gwen can generete JSON reports enabling you to feed evaulation results into other Gherkin report formatters (such as Donut) or your own custom one.

Generate JSON#

  • Standalone
  • Workspace
  • JS Project
  • Linux
  • Windows

To generate JSON reports only

./gwen -b -r output/reports -f json features/todo

To generate HTML and JSON reports

./gwen -b -r output/reports -f html,json features/todo

To generate JSON reports only

gwen -b -r output\reports -f json features\todo

To generate HTML and JSON reports

gwen -b -r output\reports -f html,json features\todo

Output

  • The JSON report files will be generated at output/reports/json/*.json. You can set this path in your Jenkins reporting configuration for example.
  • The HTML report will be generated at output/reports/index.html.
Deprecated

Gwen workspaces are deprecated in favor of JS projects coming to Gwen 3.

To generate JSON reports only

./gwen -b -f json features/todo

To generate HTML and JSON reports

./gwen -b -f html,json features/todo

To generate JSON reports only

gwen -b -f json features\todo

To generate HTML and JSON reports

gwen -b -f html,json features\todo

Output

  • The JSON report files will be generated at target/reports/json/*.json. You can feed these to other reporters.
  • The HTML report will be generated at target/reports/index.html.
