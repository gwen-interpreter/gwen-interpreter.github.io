# HTML Reports

Pretty HTML reports capture evaluated results and statistics.

## View report

Opening the generated HTML report in your browser will display the summary.

![Example HTML report](/img/reports/sample-html-report-summary.png)

Clicking the feature link in the summary will open the feature detail.

![Example HTML report](/img/reports/sample-html-report-detail.png)

### Expand Step Defs

[StepDefs](/docs/meta-specs#stepdefs) are inlined and can be expanded by hovering over steps with the mouse and clicking them in the report.

![Example HTML report](/img/reports/sample-html-report-expanded.png)
