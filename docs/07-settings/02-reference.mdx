# Settings Reference

## Core Settings

<details id="gwen-assertion-mode">
  <summary class="setting">

  ```properties
  gwen.assertion.mode
  ```
  <p>Controls whether or not hard, soft or sustained assertions are enabled by default.</p>
  </summary>
  <p>Supported values</p>
  <ul>
    <li><code>hard</code> to halt feature execution on the first assertion error and report failure (default)</li>
    <li><code>soft</code> to collect all assertion errors and continue execution whilst reporting failures</li>
    <li><code>sustained</code> to collect all assertion errors and continue execution without reporting failures</li>
  </ul>
  <Tabs
  groupId="settings"
  defaultValue="conf"
  values={[
    {label: '.conf', value: 'conf'},
    {label: '.json', value: 'json'},
    {label: '.properties', value: 'properties'}
  ]}>

  <TabItem value="conf">

  Default value

  ```json
    gwen {
      assertion {
        mode = "hard"
      }
    }
  ```

  </TabItem>
  <TabItem value="json">

  Default value

  ```json
    {
      "gwen": {
        "assertion": {
          "mode": "hard"
        }
      }
    }
  ```

  </TabItem>
  <TabItem value="properties">

  Default value

  ```properties
    gwen.assertion.mode = hard
  ```

  </TabItem>

  </Tabs>

<details id="gwen-associative-meta">
  <summary class="setting">

  ```properties
  gwen.associative.meta
  ```
  <p>Controls whether or not same named feature and meta files in a directory are to be exclusively associated.</p>
  </summary>
  <p>Supported values</p>
  <ul>
    <li><code>true</code> to enable association (default)
      <ul>
        <li>Gwen will load a discovered meta file only if it has the same name and is in the same directory as the current feature being executed.</li>
      </ul>
    </li>
    <li><code>false</code> to disable association
      <ul>
        <li>Gwen will unconditionally load every meta file discovered in the path to the current feature being executed.</li>
      </ul>
    </li>
  </ul>
  <Tabs
  groupId="settings"
  defaultValue="conf"
  values={[
    {label: '.conf', value: 'conf'},
    {label: '.json', value: 'json'},
    {label: '.properties', value: 'properties'}
  ]}>

  <TabItem value="conf">

  Default value since <Link to="/docs/gwen3">Gwen 3</Link>

  ```json
    gwen {
      associative {
        meta = true
      }
    }
  ```

  Default value in prior versions

  ```json
    gwen {
      associative {
        meta = false
      }
    }
  ```

<details id="gwen-behavior-rules">
  <summary class="setting">

  ```properties
  gwen.behavior.rules
  ```
  <p>Controls whether strict or lenient rules around Given-When-Then usage will be enforced for features.
    <br/><br/><em>Note</em>: <code>gwen.behaviour.rules</code> is no longer a supported alias (since v3.0.0)
  </p>
  </summary>
  <p>Supported values</p>
  <ul>
    <li><code>strict</code> to enforce the following strict rules (default):
      <ul>
        <li>Steps in scenarios and backgrounds must satisfy Given-When-Then order</li>
        <li>Givens must set context</li>
        <li>Whens must perform actions</li>
        <li>Thens must assert expectations</li>
        <li>Buts must perform assertions (and are permitted after Then steps)</li>
        <li>StepDefs tagged with
          <ul>
            <li><code>@Context</code> : can only be called by Givens</li>
            <li><code>@Action</code> : can only be called by Whens</li>
            <li><code>@Assertion</code> : can only be called by Thens or Buts</li>
          </ul>
        </li>
      </ul>
    </li>
    <li><code>lenient</code> to relax all strict rules</li>
  </ul>
  <Tabs
  groupId="settings"
  defaultValue="conf"
  values={[
    {label: '.conf', value: 'conf'},
    {label: '.json', value: 'json'},
    {label: '.properties', value: 'properties'}
  ]}>

  <TabItem value="conf">

  Default value since <Link to="/docs/gwen3">Gwen 3</Link>

  ```json
    gwen {
      behavior {
        rules = "strict"
      }
    }
  ```

  Default value in prior versions

  ```json
    gwen {
      behavior {
        rules = "lenient"
      }
    }
  ```

  </TabItem>
  <TabItem value="json">

  Default value since <Link to="/docs/gwen3">Gwen 3</Link>

  ```json
    {
      "gwen": {
        "behavior": {
          "rules": "strict"
        }
      }
    }
  ```

  Default value in prior versions

  ```json
    {
      "gwen": {
        "behavior": {
          "rules": "lenient"
        }
      }
    }
  ```

  </TabItem>
  <TabItem value="properties">

  Default value since <Link to="/docs/gwen3">Gwen 3</Link>

  ```properties
    gwen.behavior.rules = strict
  ```

  Default value in prior versions
  
  ```properties
    gwen.behavior.rules = lenient
  ```

  </TabItem>

  </Tabs>
