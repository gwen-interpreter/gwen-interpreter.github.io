<details id="i-should-have-1-open-tab-or-window">
  <summary class="dsl">

  ```gherkin
  I should have 1 open <tab|window>
  ```
  <p>Checks that only one tab or window is currently open. Reports an assertion error if not.</p>
  </summary>

  <p>

  <p>With custom assertion error message:</p>

  ```gherkin
  I should have 1 open <tab|window>  @Message("my custom assert fail message")
  ```
  
  </p>

  <p>Example</p>

  ```gherkin {2}
    When I navigate to "https://todomvc.com/examples/react/dist"
    Then I should have 1 open window
  ```

<details id="i-should-have-count-open-tabs-or-windows">
  <summary class="dsl">

  ```gherkin
  I should have <count> open <tab|window>s
  ```
  <p>Checks that a given number of tabs or windows are currently open. Reports an assertion error if not.</p>
  </summary>

  <p>

  <p>With custom assertion error message:</p>

  ```gherkin
  I should have <count> open <tab|window>s  @Message("my custom assert fail message")
  ```

  </p>

  <p>Where</p>
  <ul>
    <li><code>count</code> the expected number of open tabs or windows</li>
  </ul>

  <p>Example</p>

  ```gherkin {4}
    When I navigate to "https://todomvc.com/examples/react/dist"
     And I start a browser for Vue
     And I navigate to "https://todomvc.com/examples/vue"
    Then I should have 2 open browsers
  ```
