<details id="i-execute-system-process">
  <summary class="dsl">

  ```gherkin
  I execute system process "<expression>"
  ```
  <p>Executes a local system process or script.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;expression&gt;</code> is the system process or script to execute</li>
  </ul>
  <p>Example</p>

  ```gherkin
    When I execute system process "echo 'Hello Gwen World'"
     And I execute system process "path/to/script.bat"
  ```

  </div>
</details>

<details id="i-execute-system-process-delimited-by-delimiters">
  <summary class="dsl">

  ```gherkin
  I execute system process "<expression>" delimited by "<delimiters>"
  ```
  <p>Executes a delimited local system process or script.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;expression&gt;</code> the delimited system process to execute
      <ul>
        <li>Delmiters are required for special cases such as when unbalanced quoted strings are passed as arguments (see example).</li>
      </ul>
    </li>
    <li><code>&lt;delmiters&gt;</code> is a regex pattern defining the delimiter(s) used in <code>&lt;expression&gt;</code></li>
  </ul>
  <p>Example</p>

  ```gherkin
    When I execute system process "echo,"quoted text " and orphan quote"" delimited by ","
  ```

    <div><Link to="#i-execute-system-process-delimited-by-delimiters">Link</Link></div>

    <div align="right">Since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v2.53.0">v2.53.0</Link></div>
  </div>
</details>
