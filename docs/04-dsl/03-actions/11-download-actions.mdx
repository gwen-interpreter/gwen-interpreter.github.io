<details id="i-download-to-file">
  <summary class="dsl">

  ```gherkin
  I download "<url>" to "<filepath>"
  ```
  <p>Downloads a resource from a URL to a file.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;url&gt;</code> is URL to the resource to download</li>
    <li><code>&lt;filepath&gt;</code> is the path of the file to download the resource to</li>
  </ul>
  <p>Example</p>

  ```gherkin
     When I download "https://example.com/file.pdf" to "output/download/file.pdf"
  ```

</details>

<details id="i-download-to-file-ref">
  <summary class="dsl">

  ```gherkin
  I download "<url>" to <filepathRef file>
  ```
  <p>Downloads a resource from a URL to a referenced file.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;url&gt;</code> is URL to the resource to download</li>
    <li><code>&lt;filepathRef file&gt;</code> is the name of the binding containing the path of the file to download the resource to</li>
  </ul>
  <p>Example</p>

  ```gherkin {2}
    Given my out file is "output/download/file.pdf"
     When I download "https://example.com/file.pdf" to my out file
  ```

</details>

<details id="i-download-current-url-to-file">
  <summary class="dsl">

  ```gherkin
  I download the current URL to "<filepath>"
  ```

  </summary>
