<details id="i-attach-filepath-as-name">
  <summary class="dsl">

  ```gherkin
  I attach "<filepath>" as "<name>"
  ```
  or since v3.0.0
  ```gherkin
  I attach "<filepath>" as <name>
  ```

  <p>Attaches a local file to the Gwen report and assigns it a name. The link to the attachment in the report will have the given name.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;filepath&gt;</code> the path to the local file to attach</li>
    <li><code>&lt;name&gt;</code> the name to assign to the file attachment</li>
  </ul>
  <p>Example</p>

  ```gherkin
     When I attach "path/to/data.csv" as "data"
  ```
  Or since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v3.0.0">v3.0.0</Link>

  ```gherkin
     When I attach "path/to/data.csv" as data
  ```

<details id="i-write-append-text-to-file">
  <summary class="dsl">

  ```gherkin
  I <write|append> "<text>" to "<filepath>" file
  ```
  <p>Writes or appends text to a file.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;write|append&gt;</code> is one of:
      <ul>
        <li><code>&lt;write&gt;</code> to overwrite entire content of file with the provided text</li>
        <li><code>&lt;append&gt;</code> to append the provided text to existing content in file</li>
      </ul>
    </li>
    <li><code>&lt;text&gt;</code> the text to write or append to the file</li>
    <li><code>&lt;filepath&gt;</code> the path to the local file to write or append the text to</li>
  </ul>
  <p>Example</p>

  ```gherkin
     When I write "automation" to "path/to/file.txt" file
  ```

  <div class="grid-3">
    <div><Link to="#i-write-append-text-to-file">Link</Link></div>
    <div align="center"></div>
    <div align="right">Since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v3.25.0">v3.25.0</Link></div>
  </div>
</details>


<details id="i-write-append-text-ref-to-file">
  <summary class="dsl">

  ```gherkin
  I <write|append> <textRef> to "<filepath>" file
  ```
  <p>Writes or appends a text reference value to a file.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;write|append&gt;</code> is one of:
      <ul>
        <li><code>&lt;write&gt;</code> to overwrite entire content of file with the provided text</li>
        <li><code>&lt;append&gt;</code> to append the provided text to existing content in file</li>
      </ul>
    </li>
    <li>
      <code>&lt;textRef&gt;</code> is the name of the binding containing the text to write or append to the file
      <ul>
        <li>Can be the name of any binding that contains or resolves to a text value, including a web element in which case the text in the element will be dynamically retrieved and used.</li>
      </ul>
    </li>
    <li><code>&lt;filepath&gt;</code> the path to the local file to write or append the text to</li>
  </ul>
  <p>Example</p>

  ```gherkin {6}
     Given my content is
           """
           Automation and robotics
           for Gherkin
           """
     When I write my content to "path/to/file.txt" file
  ```

    <div align="right">Since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v3.25.0">v3.25.0</Link></div>
  </div>
</details>
