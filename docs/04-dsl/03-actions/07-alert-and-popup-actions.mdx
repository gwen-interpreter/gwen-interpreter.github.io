<details id="i-accept-the-alert-or-confirmation-popup">
  <summary class="dsl">

  ```gherkin
  I accept the <alert|confirmation> popup
  ```
  <p>Accepts and closes an alert or confirmation popup by clicking OK.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;alert|confirmation&gt;</code> is either:
      <ul>
        <li><code>alert</code> for an alert box with an OK button</li>
        <li><code>confirmation</code> for a confirmation box with OK and Cancel buttons</li>
      </ul>
    </li>
  </ul>
  <p>Example</p>

  ```gherkin {4}
    Given the alert link can be located by id "alert"
     When I navigate to "https://www.selenium.dev/selenium/web/alerts.html"
      And I click the alert link
      And I accept the alert popup
  ```

      </ul>
    </li>
  </ul>

  ```gherkin {4}

      And I click the confirmation link
      And I capture the confirmation popup message as the alert message
  ```
