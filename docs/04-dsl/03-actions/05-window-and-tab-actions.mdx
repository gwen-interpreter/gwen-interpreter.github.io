<details id="i-have-an-open-browser">
  <summary class="dsl">

  ```gherkin
  I have an open browser
  ```
  <p>Opens a browser if none are open</p>

<details id="i-close-the-current-browser">
  <summary class="dsl">

  ```gherkin
  I close the[ current] browser
  ```
  <p>Closes the current browser session (or does nothing if one is not open).</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>[ current]</code> including the word <code>current</code> is optional and makes no difference, since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v1.3.0">v1.3.0</Link></li>
  </ul>
  <p>Example</p>

  ```gherkin {2,4}
    When I navigate to "https://todomvc.com/examples/react/dist"
     And I close the current browser
     And I navigate to "https://todomvc.com/examples/vue/"
     And I close the browser
  ```

<details id="i-start-a-browser-for-name">
  <summary class="dsl">

  ```gherkin
  I start a browser for <name>
  ```
  <p>Starts a new browser session with the given name (closes any existing browser of that name first).</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;name&gt;</code> is a name assigned to the browser session</li>
  </ul>
  <p>Example</p>

  ```gherkin {1,3}
    When I start a browser for React
     And I navigate to "https://todomvc.com/examples/react/dist"
     And I start a browser for Vue
     And I navigate to "https://todomvc.com/examples/vue/"
    Then I should have 2 open browsers
  ```

<details id="i-switch-to-name">
  <summary class="dsl">

  ```gherkin
  I switch to <name>
  ```
  <p>Switches to a named browser session (starts a new browser if one of that name is not already open). All browser actions performed after this will execute in the named session.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;name&gt;</code> is the name of the browser session to switch to</li>
  </ul>
  <p>Example</p>

  ```gherkin {5}
    When I start a browser for React
     And I navigate to "https://todomvc.com/examples/react/dist"
     And I start a browser for Vue
     And I navigate to "https://todomvc.com/examples/vue/"
     And I switch to React
    Then the current URL should contain "react"
  ```

<details id="i-start-a-new-browser-tab-or-window">
  <summary class="dsl">


  ```gherkin {2}
     And I start a new browser tab
     And I navigate to "https://todomvc.com/examples/react/dist"
  ```

<details id="i-switch-to-tab-or-window-occurrence">
  <summary class="dsl">

  ```gherkin
  I switch to <tab|window> <occurrence>
  ```
  <p>Switches to a previously opened tab or window occurrence.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;tab|window&gt;</code> is either <code>tab</code> or <code>window</code></li>
    <li>
      <code>&lt;occurrence&gt;</code> is the tab or window occurrence to switch to (1st opened is occurrence 1, 2nd is 2, ..)
      <ul>
        <li><em>Note:</em> the occurrence number of the default tab/window in a new browser session is 0</li>
      </ul>
    </li>
  </ul>
  <p>Example</p>

  ```gherkin {4}
    When I navigate to "https://google.com"
     And I start a new browser tab
     And I navigate to "https://todomvc.com/examples/react/dist"
     And I switch to window 1
    Then the page title should contain "Google"
  ```

  <div class="grid-3">
    <div><Link to="#i-switch-to-tab-or-window-occurrence">Link</Link></div>
    <div align="center"></div>
    <div align="right">Since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v3.0.0">v3.0.0</Link></div>
  </div>
</details>

<details id="i-close-tab-or-window-occurrence">
  <summary class="dsl">

  ```gherkin
  I close <tab|window> <occurrence>
  ```
  <p>Closes a previously opened tab or window occurrence.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;tab|window&gt;</code> is either <code>tab</code> or <code>window</code></li>
    <li>
      <code>&lt;occurrence&gt;</code> is the tab or window occurrence to close to (1st opened is occurrence 1, 2nd is 2, ..)
      <ul>
        <li><em>Note:</em> you cannot close the default tab/window opened by starting a new browser session with this DSL. Use <Link to="#i-close-the-current-browser"><code>I close the[ current] browser</code></Link> instead in that case.</li>
      </ul>
    </li>
  </ul>
  <p>Example</p>

  ```gherkin {4}
    When I navigate to "https://google.com"
     And I start a new browser tab
     And I navigate to "https://todomvc.com/examples/react/dist"
     And I close tab 1
    Then the page title should contain "Google"
  ```
  
  If used in conjunction with <Link to="/docs/try-steps"><code>@Try</code></Link>, will switch to the parent window regardless of whether or not a tab or window was closed (since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v3.12.4">3.12.4</Link>).

<details id="i-send-keys">
  <summary class="dsl">

  ```gherkin
  I send "<keys>"
  ```
  <p>Sends a sequence of keys to the browser (for emulating keyboard shortcuts).</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;keys&gt;</code> is a <code>,</code> or <code>+</code> (since <Link to="https://github.com/gwen-interpreter/gwen-web/releases/tag/v2.53.0">v2.53.0</Link>) separated list of keys to send.
      <ul>
        <li>Keys can be single keyboard characters or any supported <Link to="https://seleniumhq.github.io/selenium/docs/api/java/org/openqa/selenium/Keys.html">key constants</Link></li>
        <li><i>Note:</i> Keys behave differently on different platforms</li>
      </ul>
    </li>
  </ul>
  <p>Example</p>

  ```gherkin {2}
     When I navigate to "https://google.com"
      And I send "COMMAND+t"
  ```

    <div><Link to="#i-send-keys">Link</Link></div>
