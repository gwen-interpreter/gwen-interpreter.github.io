<details id="i-switch-to-the-default-content">
  <summary class="dsl">

  ```gherkin
  I switch to the default content
  ```
  <p>Switches to either the first frame on the page, or the main document when a page contains iframes.</p>
  </summary>
  <p>Example</p>

  ```gherkin {2}
    When I navigate to "https://page-with-iframes"
     And I switch to the default content
  ```

<details id="i-switch-to-frame-content">
  <summary class="dsl">

  ```gherkin
  I switch to <frame> content
  ```
  <p>Switches the context to a frame.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;frame&gt;</code> is a frame binding name
    </li>
  </ul>
  <p>Example</p>

  ```gherkin {3}
   Given the frame can be located by id "frame_id"
    When I navigate to "https://page-with-iframes"
     And I switch to the frame content
  ```
