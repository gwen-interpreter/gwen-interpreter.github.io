<details id="i-navigate-to-url">
  <summary class="dsl">

  ```gherkin
  I navigate to "<url>"
  ```

  <p>Navigates the browser to the given URL. Starts a new browser window if one is not open.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;url&gt;</code> is the URL to navigate to (can be in <Link to="/docs/doc-strings">DocString</Link> position)</li>
  </ul>
  <p>Example</p>

  ```gherkin
    When I navigate to "https://todomvc.com/examples/react/dist"
  ```
