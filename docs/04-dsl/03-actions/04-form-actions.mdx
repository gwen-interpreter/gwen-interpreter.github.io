<details id="i-submit-form">
  <summary class="dsl">

  ```gherkin
  I submit <form>
  ```
  <p>Submits a web form.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;form&gt;</code> is the name of the form to submit</li>
  </ul>
  <p>Example</p>

  ```gherkin {3}
    Given the form can be located by name "contactform"
     When I navigate to "https://automationintesting.com/selenium/testpage"
      And I submit the form
  ```
