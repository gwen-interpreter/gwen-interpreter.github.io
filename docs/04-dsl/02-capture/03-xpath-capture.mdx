<details id="i-capture-the-node-type-in-xml-ref-by-xpath-expression-as-name">
  <summary class="dsl">

  ```gherkin
  I capture the <nodeType> in <xmlRef> by xpath "<expression>" as <name>
  ```
  <p>Captures XML content from a referenced XML string using the given XPath expression and binds it to the given name in the global scope.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;nodeType&gt;</code> is one of:
      <ul>
        <li><code>text</code> to capture the text node</li>
        <li><code>node</code> to capture a single node</li>
        <li><code>nodeset</code> to capture multiple nodes</li>
      </ul>
    </li>
    <li>
      <code>&lt;xmlRef&gt;</code> is the name of the binding containing the XML text
      <ul>
        <li>Can be the name of any binding that contains or resolves to an XML text value, including a web element in which case the text in the element will be dynamically retrieved and used.</li>
      </ul>
    </li>
    <li><code>&lt;expression&gt;</code> is the XPath selector expression</li>
    <li><code>&lt;name&gt;</code> is the name to bind the captured value to</li>
  </ul>
  <p>Example</p>

  ```gherkin {2}
    Given the xml is "<result><status>passed</status></result>"
     When I capture the text in the xml by xpath "//result/status" as the status
     Then the status should be "passed"
  ```

<details id="i-capture-the-node-type-in-element-by-xpath-expression-as-name">
  <summary class="dsl">

  ```gherkin
  I capture the <nodeType> in <element> by xpath "<expression>" as <name>
  ```
  <p>Captures XML content from a web element using the given XPath expression and binds it to the given name in the global scope.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li>
      <code>&lt;nodeType&gt;</code> is one of:
      <ul>
        <li><code>text</code> to capture the text node</li>
        <li><code>node</code> to capture a single node</li>
        <li><code>nodeset</code> to capture multiple nodes</li>
      </ul>
    </li>
    <li><code>&lt;element&gt;</code> is the name of the element containing the XML content</li>
    <li><code>&lt;expression&gt;</code> is the XPath selector expression</li>
    <li><code>&lt;name&gt;</code> is the name to bind the captured value to</li>
  </ul>
  <p>Example</p>

  ```gherkin {3}
    # assuming a web element contains xml: <result><status>passed</status></result>
    Given the xml web element can be located by id "xml"
     When I capture the text in the xml web element by xpath "//result/status" as the status
     Then the status should be "passed"
  ```
