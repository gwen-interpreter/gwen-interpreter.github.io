<details id="i-capture-pdf-text-from-the-current-url">
  <summary class="dsl">

  ```gherkin
  I capture the PDF text from the current URL
  ```
  <p>Captures the text in a PDF document from the current URL in the browser and binds it to <code>the PDF text</code> variable in the global scope.</p>
  </summary>
  <p>Example</p>

  ```gherkin {3}
    Given the pdf url is "https://example.com/file.pdf"
     When I navigate to "${the pdf url}"
      And I capture the PDF text from the current URL
     Then the PDF text should contain "Some expected text"
  ```
