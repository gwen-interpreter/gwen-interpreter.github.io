<details id="i-capture-the-current-url">
  <summary class="dsl">

  ```gherkin
  I capture the current URL
  ```
  
  <p>Captures the current browser URL and binds it as <code>the current URL</code> in the global scope.</p>
  </summary>
  <p>Example</p>

  ```gherkin {1}
    When I capture the current URL
     And I navigate to "${the current URL}?q=gwen"
  ```

<details id="i-capture-the-current-url-as-name">
  <summary class="dsl">

  ```gherkin
  I capture the current URL as <name>
  ```
  
  <p>Captures the current browser URL and binds it to the given name in the global scope.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;name&gt;</code> is the name to bind the captured URL to</li>
  </ul>
  <p>Example</p>

  ```gherkin {1}
    When I capture the current URL as the base URL
     And I navigate to "${the base URL}?q=gwen"
    Then the current URL should start with the base URL
  ```

</details>

<details id="i-capture-the-text-in-the-current-url-by-regex-as-name">
  <summary class="dsl">

  ```gherkin
  I capture the text in the current URL by regex "<expression>" as <name>
  ```

  <p>Extracts a substring from the current browser URL using the given regex and binds it to the given name in the global scope.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;expression&gt;</code> is the regex expression</li>
    <li><code>&lt;name&gt;</code> is the name to bind the captured substring to</li>
  </ul>
  <p>Example</p>

  ```gherkin {1}
    When I capture the text in the current URL by regex "q=([^&]*)" as q
    Then q should be "gwen"
  ```
