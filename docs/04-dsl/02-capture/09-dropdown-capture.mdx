<details id="i-capture-dropdown">
  <summary class="dsl">

  ```gherkin
  I capture <dropdown> <text|value>
  ```
  <p>Captures the selected text or value of a dropdown element and binds it to the element name in the global scode.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;dropdown&gt;</code> is the name of the dropdown element to capture the text or value of</li>
    <li>
      <code>&lt;dropdown&gt;</code> is one of:
      <ul>
        <li><code>text</code> to captured the displayed text of the selection</li>
        <li><code>value</code> to captured the option value of the selection</li>
      </ul>
    </li>
  </ul>
  <p>Example</p>

  ```gherkin {3}
    Given the age dropdown can be located by id "challenger_age"
     When I navigate to "https://challengers.flood.io/step/2"
      And I capture the age dropdown text
  ```
