<details id="i-capture-the-current-screenshot">
  <summary class="dsl">

  ```gherkin
  I capture the current screenshot
  ```
  <p>Captures the current screenshot and attaches it to the Gwen report.</p>
  </summary>
  <p>Example</p>

  ```gherkin {4}
    Given the todo field can be located by class "new-todo"
     When I navigate to "https://todomvc.com/examples/react/dist"
      And I enter "Walk the dog" in the todo field
      And I capture the current screenshot
  ```

<details id="i-capture-the-current-screenshot-as-name">
  <summary class="dsl">

  ```gherkin
  I capture the current screenshot as <name>
  ```
  <p>Captures the current screenshot, attaches it to the Gwen report, and binds the filepath of the saved image to the given name in the global scope.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;name&gt;</code> the name to bind the captured image filepath to</li>
  </ul>
  <p>Example</p>

  ```gherkin {4}
    Given the todo field can be located by class "new-todo"
     When I navigate to "https://todomvc.com/examples/react/dist"
      And I enter "Walk the dog" in the todo field
      And I capture the current screenshot as the screenshot image
     Then the screenshot image should contain "path/to/image.png"
  ```
