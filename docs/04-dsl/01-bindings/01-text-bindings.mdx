<details id="name-is-value">
  <summary class="dsl">

  ```gherkin
  <name> is "<value>"
  ```
  <p>Binds the given value to a name in the global scope.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;name&gt;</code> is the name to bind the value to</li>
    <li><code>&lt;value&gt;</code> is the value to bind (can be in <Link to="/docs/doc-strings">DocString</Link> position)</li>
    
  </ul>
  <p>Example</p>

  ```gherkin
    Given my name is "Gwen"
  ```

<details id="name-is-literal">
  <summary class="dsl">

  ```gherkin
  <name> is <blank|empty|true|false>
  ```
  <p>Binds a literal value to a name in the global scope.</p>
  </summary>
  <p>Where</p>
  <ul>
    <li><code>&lt;name&gt;</code> is the name to bind the literal value to</li>
    
  </ul>
  <p>Example</p>

  ```gherkin
    Given my name is blank
  ```
