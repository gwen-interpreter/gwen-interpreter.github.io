About our Logo

The Gwen logo consists of a Lorenz attractor that looks like a butterfly.

Gwen evaluates software applications by mapping Gherkin steps to instructions that:

  • Create context (Givens)
  • Perform actions (Whens)
  • Assert outcomes (Thens)

The cumulative result is a chain of instructions that execute in sequence to model cause and effect. If a change to the system disrupts the chain, then Gwen will detect the chaos that emerges.

Logo developed by Nascent Creative.
